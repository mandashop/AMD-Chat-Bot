# 아만다 챗봇 (Amanda Chatbot)

이 프로젝트는 텔레그램 그룹 채팅방 관리를 위한 종합 챗봇입니다. 출석체크, 채팅 통계, 환율/가상화폐 변환, 그리고 강력한 관리자 기능(도배 방지, 금칙어, 예약 메시지, 통계 내보내기 등)을 제공합니다. Render와 같은 클라우드 서비스에 배포하기 적합하도록 UptimeRobot용 모니터링 엔드포인트도 내장되어 있습니다.

## 주요 기능

- **채팅 통계 및 랭킹**: 유저별 메시지 수를 카운트하고 랭킹 제공 (`/stats`, `/rank`, `/mystats`)
- **출석체크**: 'ㅊㅊ', '출첵', '출석체크' 등의 키워드로 하루 1회 출석 처리 및 랭킹 조회 (`/attend`, `/attendrank`)
- **환율 및 가상화폐 변환**: 메시지 내 지정된 포맷(`KRW40000>USDT`, `BTC0.1>KRW` 등)을 인식해 실시간 환율 적용 결과 응답
- **관리자 전용 기능 (1:1 채팅)**: `/admin` 명령어로 접속하는 인라인 키보드 메뉴
  - 도배 방지 (동일 메시지 반복 횟수 및 시간 설정)
  - 닉네임 변경 감지 및 알림
  - 금칙어 설정 및 필터링
  - 예약 메시지 발송 및 반복 메시지 설정 (시간, 요일 지정)
  - 통계 데이터 초기화 및 백업 (`.TXT` 다운로드)
- **자동화 스케줄러**:
  - 매일 자정 출석 초기화
  - 매월 말일 채팅/출석 랭킹 자동 공지 및 초기화
  - 탈퇴한 계정 그룹 추방 작업 스케줄링
- **웹 서버 모니터링**: `/ping` 엔드포인트를 통한 UptimeRobot 호환 상태 모니터링

## 요구 사항

- Python 3.10+
- 텔레그램 봇 토큰 (BotFather에서 발급)

## 설치 및 실행 방법

1. **저장소 클론 및 패키지 설치**
   ```bash
   pip install -r requirements.txt
   ```

2. **환경 변수 설정**
   `.env.template` 파일을 복사하여 `.env` 파일을 생성하고, 필요한 값들을 입력합니다.
   ```bash
   cp .env.template .env
   ```
   - `BOT_TOKEN`: 텔레그램 봇 토큰
   - `ADMIN_ID`: 봇을 관리할 본인의 텔레그램 유저 ID
   - `GROUP_CHAT_ID`: 봇이 활동할 그룹 채팅방 ID (마이너스로 시작)
   - `PORT`: 웹 서버 포트 (기본값: 8080)

3. **봇 실행**
   ```bash
   python bot.py
   ```

## 배포 가이드 (Render)

1. Render에서 **Web Service**를 생성합니다.
2. Build Command: `pip install -r requirements.txt`
3. Start Command: `python bot.py`
4. Environment Variables에서 `.env`에 해당하는 값들을 설정해 줍니다.
5. 배포 완료 후 제공된 URL 뒤에 `/ping`을 붙여 UptimeRobot에 등록하면 봇이 24시간 깨어있도록 유지할 수 있습니다.
